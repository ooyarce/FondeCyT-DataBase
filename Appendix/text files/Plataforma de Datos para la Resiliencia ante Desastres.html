play_arrowDescripción

En este conjunto de datos, se puede ver los registros de aceleración de terremotos entre 1994 a 2009 en Chile. Estos registros son de RENADIC, la red de cobertura nacional de acelerógrafos. Los registros no están corregidos. La información disponible incluye el nombre de estación, la ubicación de terremoto, la fecha, la profundidad, la magnitud, el numero de identificación, y los registros.

import pytrend
import pandas as pd

session = pytrend.itrend_developer_tools()

session.set_credentials(
    access_key_id = '',    # Ingresa tu access_key_id
    secret_access_key = '' # Ingresa tu secret_access_key
)

# Conjunto de datos
dataset_id = 'itrend-ds:k2dimntt23gm0b7t'
collection_id = 'Identificador'

# Obtener formatos disponibles
dataset_formats = session.get_dataset_formats(dataset_id)
fmt = dataset_formats[0] # Escoger el formato que más le acomode: dataset_formats = [csv, shp, geojson, xlsx]

# Descargar archivo
response = session.download_file(dataset_id, fmt)

# Descargar un elemento
filename = response.get('filename')
delimiter = response.get('delimiter')
df = pd.read_csv(filename, delimiter)

element_formats = session.get_element_formats(dataset_id)
efmt = element_formats[0] # Escoger el formato que más le acomode: element_formats = [plainText, npz, mat]

for r, row in df.iterrows():
    element_id = row[collection_id]
    element_response = session.download_file(dataset_id, efmt, element_id)
    break
